AMQP settings object provides server status information::

    >>> from cone.mdb.amqp import consumer
    >>> from zamqp.runner import cleanup
    
    >>> consumer()
    True
    
    >>> amqp.amqpstatus
    'OK'
    
    >>> cleanup()
    
    >>> amqp.amqpstatus
