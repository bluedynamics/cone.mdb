Solr Settings form
==================

Check ACL::

    >>> import cone.mdb.browser.solr
    >>> from cone.tile import render_tile
    >>> from pyramid.testing import DummyRequest
    >>> from cone.app import root
    >>> solr = root['settings']['solr']
